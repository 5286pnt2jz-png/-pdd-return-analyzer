店铺商品导出
操作步骤：
1. 打开店铺主页，点击「全部商品」标签
2. 向下滚动加载更多商品
3. 点击下方「采集当前页面商品」
4. 点击「导出为CSV文件」
提示： 包含商品ID、名称、价格、销量等信息
商家商品活动价导出
操作步骤：
1. 打开商家后台 → 商品管理 → 商品列表
2. 筛选需要导出的商品
3. 点击下方「采集当前页面商品」
4. 点击「导出为CSV文件」
提示： 包含商品ID、名称、原价、活动价、到手价等信息
